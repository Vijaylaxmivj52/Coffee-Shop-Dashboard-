 2. Short Description / Purpose

This project is an **interactive Power BI dashboard** built to analyze the performance of a coffee shop.  
The goal is to help stakeholders quickly understand:

- How sales are performing over time  
- Which products and locations are most profitable  
- Customer purchasing patterns (time of day, category, etc.)  

It is designed as a **beginner-friendly data analytics project** to showcase skills in **data cleaning, modeling, and visualization** using Power BI.
